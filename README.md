# pretty-device-name
 This library can easily extract prettier device name

## Install

```bash
npm install pretty-device-name
```

## Usage

```ts
import {getDeviceName} from "pretty-device-name"

const deviceName = getDeviceName()
// returns Hedo's MacBook Pro
```
